# Chi Evolution

Sistema de memoria vectorial y voz para asistentes de IA.

## Componentes

- `chi-memory.js` - Sistema de memoria vectorial con LanceDB
- `chi-voice.js` - Integración TTS/STT (Fish Audio + Deepgram)
- `test-memory.js` - Tests de integración
